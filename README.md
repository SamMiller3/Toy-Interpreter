# Toy Interpreter

A toy interpreter I am currently writing in python 

Commands include: 

```
print()
input()
if{}
else{}
goto
int <variable name> = <data>
str <variable name> = <data>
flt <variable name> = <data>
```

Features:
```
* expression handling respecting precedence of operations
* nested ifs
* functions w/ recursion
* Static typed
```
Data types:

```
* Integers
* Floata
* Strings
```
